# Example Images

Place 3-5 sample dermoscopic images here for the Gradio interface.

## Required Files:
- `example1.jpg` or `.png`
- `example2.jpg` or `.png`
- `example3.jpg` or `.png`

## Where to Get Images:

### Option 1: From Your Experiments
Download sample images from your Google Drive experiments folder

### Option 2: ISIC Dataset
Use test images from the ISIC 2018 dataset

### Option 3: Public Dermoscopy Images
Search for "dermoscopic skin lesion images" (ensure they're public domain)

## Image Requirements:
- Format: JPG or PNG
- Size: Any (will be resized to 256x256)
- Content: Dermoscopic skin images
- Recommended: Clear lesion boundaries for best demo

## Note:
These images will appear as clickable examples in the Gradio interface, allowing users to quickly test the model without uploading their own images.
